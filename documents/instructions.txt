To create a new database and user account
=========================================

create database quiz;
create user quiz_dev;
grant all on quiz.* to 'quiz_dev'@'localhost' identified by 'quiz_dev';
flush privileges;

To Bootup the application
=========================
Export an environment variable
set QUIZ_DATA_FOLDER="C:\Abhinav\Quiz\quiz_poc\documents\questions"
set QUIZ_CONNECTION_STRING="jdbc:mysql://localhost:3306/quiz"
set QUIZ_CONNECTION_USER="quiz_dev"
set QUIZ_CONNECTION_PASSWORD="quiz_dev"
set ANDROID_HOME="C:\Abhinav\Tools\Android\sdk"

And then execute the following command from quiz-ws module:
mvn spring-boot:run

To install ionic
================
 npm install -g cordova ionic
 ionic start myApp tabs
 cd myApp
 ionic platform add android
 ionic build android
 ionic emulate android
 ionic serve
